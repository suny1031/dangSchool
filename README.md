# 댕스쿨   
kh정보교육원 semi project

**"바쁠때도 나 대신 강아지를 케어해줄 곳이 필요해"**   
**"우리집 주변에도 강아지 유치원이 있나?"**   
**"유치원에서 내 반려동물은 뭘 할까?"**   

댕스쿨과 함께라면 가능합니다. 견주와 애견유치원의 소통을 제공해주는 웹 사이트


## 1. 개발 환경
운영체제 : windows OS   
개발도구 :  eclipse   
DBMS : Oracle   
Server : Apache Tomcat 8.5   
Language : Java   

## 2. 구현 기능
로그인 기능, 회원정보 관리 기능, 게시판 기능, 위치확인 기능, 후기 기능, 예약 기능, 캘린더 기능, 알림 기능
